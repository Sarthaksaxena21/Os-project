# Os-project
Energy-Efficient CPU Scheduling Algorithm Develop a CPU scheduling algorithm that reduces energy consumption without compromising performance. The design integrates DVFS, thermal-aware scheduling, and workload prediction to optimize task placement on heterogeneous cores. It ensures balanced performance, fairness, and extended battery life.
This project implements an Energy-Efficient CPU Scheduling Algorithm (EECSA) designed to reduce power consumption while maintaining system performance. The algorithm integrates:

Dynamic Voltage and Frequency Scaling (DVFS)
Thermal-aware scheduling
Workload prediction
Heterogeneous core utilization (big + little cores)

The system is implemented in C++ and evaluated using simulation, with results visualized using Python .
